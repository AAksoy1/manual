PontLista[ <Lista> ]::
  A két elemet tartalmazó listák listájából létrehoz egy pontokat tartalmazó listát.

[EXAMPLE]
====

`++PontLista[{{1,2},{3,4}}]++` eredménye \{(1,2),(3,4)}.

====

ÚtParaméter[ <Az út egy ponja> ]::
  Megadja az út egy kiválasztott pontjának paraméterét (0 és 1 között).

[EXAMPLE]
====

Legyen `++f(x) = x² + x - 1++` egy függvény és `++A = (1, 1)++` egy pont a függvényen. Az `++ÚtParaméter[A]++` parancs
eredménye _a = 0.47_.

====

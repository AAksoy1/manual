== CAS nézet

SkalárisSzorzat[ <Vektor>, <Vektor> ]::
  Megadja a két vektor skaláris szorzatát.

[EXAMPLE]
====

`++SkalárisSzorzat[{1, 3, 2}, {0, 3, -2}]++` eredménye _5_.

====

[NOTE]
====

*Jegyzet:*

Lásd xref:/commands/VektoriálisSzorzat.adoc[Vektoriális szorzat].

====

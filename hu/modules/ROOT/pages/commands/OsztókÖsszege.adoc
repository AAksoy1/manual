OsztókÖsszege[ <Szám> ]::
  Megadja a szám összes pozitív osztójának az összegét, beleértve az adott számot is.

[EXAMPLE]
====

`++OsztókÖsszege[15]++` eredménye _24_ (_1 + 3 + 5 + 15_).

====

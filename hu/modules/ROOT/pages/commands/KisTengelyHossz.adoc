KisTengelyHossz[ <Kúpszelet> ]::
  Meghatározza a kúpszelet melléktengelyének hosszának felét.

[EXAMPLE]
====

`++ KisTengelyHossz[x^2 + 2y^2 - 2x - 4y = 5]++` eredménye _2_.

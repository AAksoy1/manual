CímkeMutatása[ <Alakzat>, <Logikai érték> ]::
  Megjeleníti vagy elrejti az alakzat címkéjét.

[EXAMPLE]
====

Legyen `++f(x) = x^2++`.

A `++ CímkeMutatása[f, true]++` parancs megjeleníti az _f_ függvény címkéjét.

====

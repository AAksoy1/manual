LineárisExcentricitás[ <Kúpszelet> ]::
  Kiszámítja egy kúpszelet excentricitását, vagyis a kúpszelet középpontja és fókuszpontja közti távolságot.

[EXAMPLE]
====

`++LineárisExcentricitás[4x^2 - y^2 + 16x + 20 = 0]++` eredménye 2.24

====

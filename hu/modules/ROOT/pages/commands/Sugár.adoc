Sugár[ <Kúpszelet> ]::
  Kiszámítja a megadott kúpszelet rádiuszát.

[EXAMPLE]
====

* Egy c (p.l. c:(x - 1)² + (y - 1)² = 9) rádiusza:

`++Radius[c]++` eredménye _a = 3_.

* `++Radius[(x - 2)² + (y - 2)² = 16]++` eredménye _a = 4_.

====

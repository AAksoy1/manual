`++Komplexszámként[(3, 2)]++` eredménye _3 + 2ί_.

== CAS nézet

Átváltja a megadott vektort vagy számot komplex számra (algebrai alakúra).

[EXAMPLE]
====

`++Komplexszámként[(3, 2)]++` eredménye _2ί + 3_.

====

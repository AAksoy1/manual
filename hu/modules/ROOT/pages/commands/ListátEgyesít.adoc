ListátEgyesít[ <Lisat> ]::
  Egyesíti a megadott listákat.

[EXAMPLE]
====

A `++ListátEgyesít[{2, 3, {5, 1}, {{2, 1, {3}}}}]++` parancs eredménye _lista1 = \{2, 3, 5, 1, 2, 1, 3}_.

Irány[ <Egyenes> ]::
  Az egyenes egy irányvektorát adja meg.

[EXAMPLE]
====

`++Irány[-2x + 3y + 1 = 0]++` eredménye az stem:[u= \begin\{pmatrix} 3 \\ 2 \end\{pmatrix} ] vektor.

====

[NOTE]
====

*Jegyzet:* Az _ax + by = c_ egyenletű egyenes irányvektora _(b, - a)_.

====

TeljesNégyzet[ <Másodfokú Függvény>]::
  Teljes négyzetté egészíti ki a másodfokú függvényt, vagyis megadja stem:[a(x-h)^2+k] alakban.

[EXAMPLE]
====

`++TeljesNégyzet[x^2 - 4x + 7]++` eredménye _1 (x - 2)^2^ + 3_.

====

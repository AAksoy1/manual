ZapisKaoListaTočaka[ <lista> ]::
  Izradit će listu točaka iz liste koja sadrži dvočlane liste kao elemente.

[EXAMPLE]
====

`++ZapisKaoListaTočaka[{{1,2},{3,4}}]++` izradit će listu točaka \{(1,2),(3,4)}.

====

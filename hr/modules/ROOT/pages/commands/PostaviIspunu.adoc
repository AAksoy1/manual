PostaviIspunu[ <objekt>, <broj> ]::
  Mijenja prozirnost ispune zadanog objekta. Broj mora biti iz intervala [0,1], gdje 0 znači posve proziran a 1 znači
  100% neproziran. Brojevi izvan ovog intervala se ignoriraju.

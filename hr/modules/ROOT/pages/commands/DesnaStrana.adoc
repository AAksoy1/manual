  Daje desnu stranu zadane jednad탑be.
DesnaStrana[ \{ <jednad탑ba> } ]::
  Daje desne strane zadanih jednad탑bi.

[EXAMPLE]
====

`++DesnaStrana[{a^2 + b^2 = c^2, x + 2 = 3x + 1}]++` daje listu _\{c^2^, 3x + 1}_.

====

[EXAMPLE]
====

`++DesnaStrana[x + 2 = 3x + 1]++` daje _3x + 1_.

====

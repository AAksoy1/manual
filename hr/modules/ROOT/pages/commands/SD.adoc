SD[ <lista brojeva> ]::
  Izračunava standardnu devijaciju brojava iz liste.

[EXAMPLE]
====

`++SD[{1, 2, 3, 4, 5}]++` daje _1,41_.

====

== CAS prikaz

SD[ <lista brojeva> ]::
  Izračunava standardnu devijaciju brojava iz liste.

[EXAMPLE]
====

`++SD[{1, 2, 3, 4, 5}]++` daje stem:[\sqrt\{2}].

====

nPr[ <broj>, <broj> ]::
  Daje broj svih varijacija bez ponavljanja od _r_ elemenata unutar skupa od _n_ elemenata (varijacije r-tog razreda od
  n elemenata).

[EXAMPLE]
====

`++nPr[10, 2]++` daje _90_.

====

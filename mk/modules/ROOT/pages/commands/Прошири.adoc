Прошири[ <функција> ]::
  Ја проширува дадената функција

[EXAMPLE]
====

`++Expand[(2x - 1)² + 2x + 3]++` дава _4x² - 2x + 4_.

====

Оваа команда работи во xref:/CAS_Поглед.adoc[CAS Поглед] исто така.

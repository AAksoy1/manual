ДвоенОднос[ <точка A>, <точка B>, <точка C>, <точка D> ]::
  Го пресметува одност λ на четири колинеарни точки _A_, _B_, _C_, and _D_, каде _λ_ = ПростОднос[B, C, D] /
  ПростОднос[A, C, D].

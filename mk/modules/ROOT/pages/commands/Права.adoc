Права[ <точка А>, <точка B> ]::
  Дава права низ дадените две точки _A_ и _B_.
Права[ <точка>, <права> ]::
  Дава права низ дадена точка која е паралелна со дадена права.
Права[ <точка>, <вектор> ]::
  Дава права низ дадена точка со насочен вектор _v_.

БукваВоУникод[ <текст> ]::
  Дава соодветната број на уникод за буквата [https://en.wikipedia.org/wiki/Unicode[Unicode]].

[EXAMPLE]
====

`++БукваВоУникод["a"]++` дава 97.

====

[NOTE]
====

Буквата мора да биде помеѓу пар дупли наводници.

====

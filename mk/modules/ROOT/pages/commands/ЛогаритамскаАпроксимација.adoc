ЛогаритамскаАпроксимација[ <листа на точки> ]::
  Ја пресметува кривата на логаритамска регресија.

[EXAMPLE]
====

`++ЛогаритамскаАпроксимација[{(ℯ,1), (ℯ^2, 4)}]++` дава _3 ln(x) - 2_.

====

[NOTE]
====

====

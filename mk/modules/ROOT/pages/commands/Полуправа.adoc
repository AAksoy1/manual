Полуправа[ <точка А>, <точка B> ]::
  Дава полуправа со почеток во точката _А_ која минува низ точка _B_.
Полуправа[ <точка>, <вектор V> ]::
  Дава полуправа со почеток во дадена точка, која има насока вектор _V_.

AyarlaDoğruStili( <Doğru>, <Sayı> )::
  Verilen nesnenin doğru stilini aşağıdaki tabloya göre değiştirir ([0,4] aralığının dışındaki sayılar 0 olarak ele
  alınır).

[cols=",",options="header",]
|===
|Sayı |Stil
|0 |Tam
|1 |Kesikli uzun
|2 |Kesikli kısa
|3 |Noktalı
|4 |Kesik-nokta
|===

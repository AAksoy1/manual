= Tavan Komut
ifdef::env-github[:imagesdir: /tr/modules/ROOT/assets/images]

Tavan( <Kuadrik> )::
  Sınırlı bir kuadriğin tavanını oluşturur.

[EXAMPLE]
====

*Örnekler:*

* `++Tavan( silindir )++` komutu bir çember verir.
* `++Tavan( koni )++` komutu koninin tepe noktasını verir.

====

====

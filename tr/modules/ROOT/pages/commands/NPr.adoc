:page-en: commands/NPr

nPr ( <Sayı n>, <Sayı r> )::
  _n_ elemanlı bir listeden _r_ elemanın olası tekrarsız permütasyonlarının sayısını verir.

[EXAMPLE]
====

`++nPr(10, 2)++` komutu _90_ sonucunu verir.

====

== CAS Söz Dizimi

nPr ( <Sayı n>, <Sayı r> )::
  _n_ elemanlı bir listeden _r_ elemanın olası tekrarsız permütasyonlarının sayısını verir.

[EXAMPLE]
====

*Örnekler:*

* `++nPr(10, 2)++` komutu _90_ sonucunu verir.
* `++nPr(n, 3)++` komutu _stem:[\frac\{n!}\{(n-3)!}]_ sonucunu verir.

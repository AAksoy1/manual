SommeDiviseurs( <Nombre> )::
  Calcule la somme de tous les diviseurs positifs du nombre, nombre y compris.

[EXAMPLE]
====

`++SommeDiviseurs(15)++` retourne _24_, la somme _1 + 3 + 5 + 15_.

====

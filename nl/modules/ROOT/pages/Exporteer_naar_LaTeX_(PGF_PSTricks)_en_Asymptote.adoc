== Beperkingen aan deze exportfuncties

Behalve het exporteren als Animated GIF, werkt het exporteren enkel voor het 2D Tekenvenster. Bovendien kan je volgende
types niet exporteren:

* impliciete krommen
* meetkundige plaatsen

== CAS Syntax

Eigenwaarden( <Matrix> )::
  Berekent de eigenwaarden van de gegeven matrix.

[EXAMPLE]
====

`++Eigenwaarden({{1, 2}, {3, 4}})++` geeft stem:[ \left\\{ \frac\{\sqrt\{33} + 5}\{2}, \frac\{-\sqrt\{33} + 5}\{2}
\right\} ]

====

SigmaYY( <Lijst met punten> )::
  Berekent de som van de kwadraten van de _y_-coördinaten van de gegeven punten.

[EXAMPLE]
====

`++SigmaYY( { (-3, 4), (-1, 4), (-2, 3), (1, 3), (2, 2), (1, 5) } )++` geeft _79_.

====

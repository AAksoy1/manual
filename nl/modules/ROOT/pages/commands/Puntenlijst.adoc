Puntenlijst( <Lijst> )::
  Creëert een puntenlijst uit een lijst van lijsten met telkens twee elementen.

[EXAMPLE]
====

`++Puntenlijst({{1,2},{3,4}})++` geeft \{(1,2),(3,4)}.

====

:page-en: commands/MajorAxis_Command

Hoofdas( <Kegelsnede> )::
  Geeft de vergelijking vn de hoofdas van de kegelsnede.

[EXAMPLE]
====

`++Hoofdas(x^2 / 9 + y^2 / 4 = 1)++` geeft _y_ = 0

====

[NOTE]
====

Zie ook het commando xref:/commands/Nevenas.adoc[Nevenas].

====

LogRegr( <Lijst van punten> )::
  Berekent de logaritmische regressiekromme in de vorm a + b ln(x).

[EXAMPLE]
====

`++LogRegr({(ℯ, 1), (ℯ^2, 4)})++` geeft _-2 + 3 ln(x)_.

====

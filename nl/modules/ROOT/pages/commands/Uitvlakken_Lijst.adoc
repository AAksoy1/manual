UitvlakkenLijst( <Lijst> )::
  Vlakt lijsten uit tot één lijst.

[EXAMPLE]
====

`++UitvlakkenLijst({2, 3, {5, 1}, {{2, 1, {3}}}})++` geeft _lijst1 = \{2, 3, 5, 1, 2, 1, 3}_.

====

Omkeren( <Lijst> )::
  Keert de volgorde van een xref:/Lijsten.adoc[lijst] om.

[EXAMPLE]
====

`++Omkeren(lijst1)++` keert `++lijst1 = {(1, 2), (3, 4), (5, 6)}++` om en creëert _lijst2 = \{(5, 6), (3, 4), (1, 2)}_

====

== CAS Venster

Omkeren( <Lijst> )::
  Keert de volgorde van een xref:/Lijsten.adoc[lijst] om.

[EXAMPLE]
====

`++Omkeren({1, 2, 3, 4})++` keert de lijst om en creëert _\{4, 3, 2, 1}_

====

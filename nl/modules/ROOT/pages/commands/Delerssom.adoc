= Delerssom Commando
:page-en: commands/DivisorsSum_Command
ifdef::env-github[:imagesdir: /nl/modules/ROOT/assets/images]

Delerssom( <Getal> )::
  Berekent de som van alle positieve delers, het getal inbegrepen.

[EXAMPLE]
====

`++Delerssom(15)++` geeft _24_, de som _1 + 3 + 5 + 15_.

====

ANOVA( <Lijst>, <Lijst>, ... )::
  Voert een http://en.wikipedia.org/wiki/Anova[ANOVA] test uit op de gegeven lijsten van getallen.
  De resultaten verschijnen in de vorm van een lijst als \{P waarde, F test}.

Hernoem( <Object>, <Naam> )::
  Verandert het xref:/Labels_en_titels.adoc[label] van een gegeven object.

[EXAMPLE]
====

`++c: x^2 + 2y^2 = 2++`.

`++Hernoem(c, ell)++` verandert het label in _ell_.

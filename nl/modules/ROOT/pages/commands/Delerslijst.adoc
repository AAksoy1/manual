Delerslijst( <Getal> )::
  Geeft de lijst van alle positieve delers, het getal zelf inbegrepen.

[EXAMPLE]
====

`++Delerslijst(15)++` geeft _\{1, 3, 5, 15}_, de lijst van alle positieve delers van _15_, _15_ inbegrepen.

====

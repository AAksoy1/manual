Puntbinnen( <Regio> )::
  Creëert een punt binnen een gegeven regio.

[EXAMPLE]
====

`++ Puntbinnen(x²+y²=4)++` creëert een punt dat je vrij kan verslepen binnen de cirkel C(0,2) maar niet daarbuiten.

MeetkundigGemiddelde( <Lijst met waarnemingsgetallen> )::
  Berekent het https://nl.wikipedia.org/Meetkundig_gemiddelde[meetkundig gemiddelde] van een gegeven lijst
  waarnemingsgetallen.

[EXAMPLE]
====

`++MeetkundigGemiddelde({13, 7, 26, 5, 19})++` geeft 11.76.

====

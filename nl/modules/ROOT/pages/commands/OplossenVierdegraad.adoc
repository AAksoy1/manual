== CAS Syntax

OplossenVierdegraad( <vierdegraadsveelterm> )::
  Lost een gegeven vierdegraadsvergelijking op en toont een lijst met alle oplossingen.

[EXAMPLE]
====

`++OplossenVierdegraad[ x^4 + x^3 + x^2 + x ]++` geeft \{0, -1, _i_, -_i_ }

====

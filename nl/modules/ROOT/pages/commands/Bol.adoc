Bol( <Punt>, <Straal> )::
  Creëert een bol met gegeven middelpunt en straal.
Bol( <Punt>, <Punt> )::
  Creëert een bol met als middelpunt het eerste punt, die door het tweede punt gaat.

[EXAMPLE]
====

`++Bol((0, 0, 0), (1, 1, 1))++` geeft _x² + y² + z² = 3_

====

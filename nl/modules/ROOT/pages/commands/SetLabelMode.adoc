SetLabelMode( <Object>, <Getal> )::
  Een getal van 0 tot 3 bepaalt de inhoud van het label van een gegeven object.

|===
|Getal |Mode
|0 |Naam
|1 |Naam + Waarde
|2 |Waarde
|3 |Titel

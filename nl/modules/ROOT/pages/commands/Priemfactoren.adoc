Priemfactoren( <Getal> )::
  Geeft de lijst van priemfactoren waarvan het product gelijk is aan het gegeven getal.

[EXAMPLE]
====

* `++Priemfactoren(1024)++` geeft _\{2, 2, 2, 2, 2, 2, 2, 2, 2, 2}_.
* `++Priemfactoren(42)++` geeft _\{2, 3, 7}_.

====

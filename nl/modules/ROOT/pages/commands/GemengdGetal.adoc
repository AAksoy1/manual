GemengdGetal( <Getal> )::
  Zet een getal om naar een gemengde breuk.

[EXAMPLE]
====

* `++GemengdGetal(3.5)++` geeft stem:[3 + \frac\{1}\{2}].
* `++GemengdGetal(12 / 3)++` geeft _4_.
* `++GemengdGetal(12 / 14)++` geeft stem:[\frac\{6}\{7}].

====

====

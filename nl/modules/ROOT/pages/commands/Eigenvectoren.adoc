== CAS Syntax

Eigenvectoren( <Matrix> )::
  Het commando berekent de eigenvectoren van de gegeven matrix.

[EXAMPLE]
====

`++Eigenvectoren({{1, 2}, {3, 4}})++` geeft stem:[ \left(\begin\{array}\sqrt\{33} - 3&-\sqrt\{33} -
3\\6&6\\\end\{array}\right) ]

====

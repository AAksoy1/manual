Deelverhouding( <Punt>, <Punt>, <Punt> )

Geeft de deelverhouding _λ_ van drie collineaire punten _A_, _B_ en _C_, waarbij _C = A + λ * AB_.

[EXAMPLE]
====

`++Deelverhoudng((-1, 1), (1, 1), (4, 1))++` geeft _2.5_

====

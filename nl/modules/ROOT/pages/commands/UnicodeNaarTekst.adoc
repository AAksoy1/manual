UnicodeNaartTekst( <Lijst van gehele getallen> )::
  Vormt de lijst met gehele Unicode getallen terug om naar tekst.

[EXAMPLE]
====

`++UnicodeNaarTekst({104, 97, 108, 108, 111})++` geeft de tekst _"hallo"_.

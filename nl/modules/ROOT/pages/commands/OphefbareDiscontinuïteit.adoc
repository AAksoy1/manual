OphefbareDiscontinuïteit( <Functie> )

Berekend de ophefbare discontinuïteit in een punt voor een gebroken rationale functie.

[EXAMPLE]
====

`++OphefbareDiscontinuïteit((3-x)/(2 x^(2)-6 x))++` geeft _(3,-0.17)_.

====

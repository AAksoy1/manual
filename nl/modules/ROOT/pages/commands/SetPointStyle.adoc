SetPointStyle( <Punt>, <Getal> )

Bepaalt (met een getal van 0 tot 9) de puntstijl van een gegeven punt volgens onderstaande tabel:

[cols=",,",options="header",]
|===
|Getal |Stijl |Symbool
|0 |Vol punt |●
|1 |Kruis |⨯
|2 |Hol punt |○
|3 |Plusteken |+
|4 |Volle ruit |◆
|5 |Holle ruit |◇
|6 |Driehoek |▲
|7 |Omgekeerde driehoek |▼
|8 |Driehoek rechts |▶
|9 |Driehoek links |◀
|===

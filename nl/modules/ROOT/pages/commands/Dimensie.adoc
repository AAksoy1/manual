Dimensie( <Object> )::
  Geeft de dimensie van een vector of een matrix.

[EXAMPLE]
====

`++Dimensie({{1, 2}, {3, 4}, {5, 6}})++` geeft _\{3, 2}_.

====

[EXAMPLE]
====

`++Dimensie({1, 2, 0, -4, 3})++` geeft _5_.

====

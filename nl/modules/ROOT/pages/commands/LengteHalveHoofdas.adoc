LengteHalveHoofdas( <Kegelsnede> )::
  Geeft de halve lengte van de hoofdas van de kegelsnede.

[EXAMPLE]
====

`++LengteHalveHoofdas((x - 1)^2 + (y - 2)^2 = 4)++` geeft _2_.

====

====

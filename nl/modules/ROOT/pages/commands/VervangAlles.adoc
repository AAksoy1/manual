Vervangen( <Tekst>, <te vervangen tekst>, <nieuwe tekst> )::
  Vervangt in een gegeven tekst de _te vervangen tekst_ door de gegeven _nieuwe tekst_.

[EXAMPLE]
====

`++Vervangen("3cos(t)+cos(2y)", "cos", "sin")++` geeft `++"3sin(t)+sin(2y)"++`.

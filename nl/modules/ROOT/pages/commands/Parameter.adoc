Parameter( <Parabool> )::
  Geeft de parameter van de parabool. Dit is de afstand tussen de richtlijn en het brandpunt.

[EXAMPLE]
====

`++Parameter(y = x^2 - 3x + 5)++` geeft 0.5

====

SetLineStyle( <Object>, <Getal> )::
  Bepaalt met een getal van 0 tot 4 de lijnstijl van het gegeven object volgens volgende tabel:

[cols=",",options="header",]
|===
|Getal |Stijl
|0 |Vol
|1 |Lange streepjes
|2 |Korte streepjes
|3 |Stippellijn
|4 |punt-streeplijn
|===

== CAS Syntax

KarakteristiekeVeelterm( <Matrix> )::
  Creëert de http://en.wikipedia.org/wiki/Characteristic_polynomial[karakteristieke veelterm] van de gegeven matrix.

[EXAMPLE]
====

`++KarakteristiekeVeelterm({{1,2},{3,4}})++` geeft stem:[x^2-5x-2].

====

TekstNaarUnicode( <Tekst> )::
  Vormt een tekst om in een lijst met Unicode getallen, een voor elk krakter.
  \{\{Example|1=
  * `++TekstNaarUnicode("Some text")++` geeft volgende lijst met Unicode getallen _\{83, 111, 109, 101, 32, 116, 101,
  120, 116}_.

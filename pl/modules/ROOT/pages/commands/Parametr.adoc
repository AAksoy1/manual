:page-en: commands/Parameter

Parametr( <Parabola> )::
  Zwraca parametr paraboli, który jest równy odległości ogniska od kierownicy paraboli.

[EXAMPLE]
====

`++Parametr(y = x^2 - 3x + 5)++` daje w wyniku _0.5_.

====

Parabola( <Punkt>, <Prosta> )::
  Zwraca parabolę o danym ognisku (punkt) i kierownicy (prosta).

[EXAMPLE]
====

Niech a = Prosta\((0,1), (2,1)). `++Parabola((3, 3), a)++` daje w wyniku _x² - 6x - 4y = -17_ .

====

====

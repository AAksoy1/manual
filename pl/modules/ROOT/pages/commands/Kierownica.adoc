Kierownica( <Stożkowa> )::
  Zwraca kierownicę krzywej stożkowej.

[EXAMPLE]
====

`++Kierownica(x^2 - 3x + 3y = 9)++` daje w wyniku prostą _y_ = 4.5

====

[NOTE]
====

Zobacz także polecenie xref:/commands/Ogniskowa.adoc[Ogniskowa] command.

====

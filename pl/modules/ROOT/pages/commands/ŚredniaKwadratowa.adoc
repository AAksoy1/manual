ŚredniaKwadratowa( <Lista Liczb> )::
  Zwraca https://pl.wikipedia.org/wiki/%C5%9Arednia_kwadratowa[średnią kwadratową] danej listy liczb.

[EXAMPLE]
====

`++ŚredniaKwadratowa({3, 4, 5, 3, 2, 3, 4})++` daje w wyniku _3.5456_.

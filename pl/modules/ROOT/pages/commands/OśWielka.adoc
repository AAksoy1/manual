OśWielka( <Conic> )::
  Zwraca równanie osi wielkiej krzywej stożkowej.

[EXAMPLE]
====

`++OśWielka(x^2 / 9 + y^2 / 4 = 1)++` zwraca _y_ = 0.

====

[NOTE]
====

Zobacz także polecenie xref:/commands/OśMała.adoc[OśMała].

====

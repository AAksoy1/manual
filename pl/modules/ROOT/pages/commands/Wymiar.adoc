Wymiar( <Obiekt> )::
  Zwraca wymiar wektora lub macierzy.

[EXAMPLE]
====

`++Wymiar({1, 2, 0, -4, 3})++` daje w wyniku _5_.

`++Wymiar({{1, 2}, {3, 4}, {5, 6}})++` daje w wyniku _{3, 2}_.

====

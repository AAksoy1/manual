MajorAxis( <Conic> )::
  Returns the equation of the major axis of the conic section.

[EXAMPLE]
====

`++MajorAxis(x^2 / 9 + y^2 / 4 = 1)++` returns _y_ = 0.

====

[NOTE]
====

See also xref:/s_index_php?title=MinorAxis_Command_action=edit_redlink=1.adoc[MinorAxis] command.

====

Delete( <Object> )::
  Deletes the object and all its dependent objects.

[EXAMPLE]
====

`++Delete(a)++` clears _a_.

====

== CAS Syntax

Delete( <Object> )::
  Deletes the object and all its dependent objects in GeoGebra and removes any value assigned to the object in the CAS.

[EXAMPLE]
====

`++Delete(a)++` clears _a_.

====

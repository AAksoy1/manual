SetLineStyle( <Line>, <Number> )::
  Changes the line style of given object according to following table (numbers out of range [0,4] are treated as 0).

[cols=",",options="header",]
|===
|Number |Style
|0 |Full
|1 |Dashed long
|2 |Dashed short
|3 |Dotted
|4 |Dash-dot
|===

== Sintassi CAS

PolinomioMinimo(Matrice)::
  Restituisce il http://en.wikipedia.org/wiki/it:Polinomio_minimo[polinomio minimo] della matrice indicata.

[EXAMPLE]
====

`++PolinomioMinimo({{1,0},{0,1}})++` restituisce stem:[x-1].

====

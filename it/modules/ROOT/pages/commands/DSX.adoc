= Comando DSX
:page-en: commands/SDX
ifdef::env-github[:imagesdir: /it/modules/ROOT/assets/images]

DSX(Lista di punti)::
  Restituisce la https://it.wikipedia.org/Deviazione_standard[deviazione standard] delle ascisse dei punti
  appartenenti alla lista indicata.

[EXAMPLE]
====

`++DSX({(1, 1), (2, 2), (3, 1), (3, 3), (4, 2), (3, -1)})++` restituisce _0.94_

====

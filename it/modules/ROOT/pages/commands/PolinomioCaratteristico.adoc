== Sintassi CAS

PolinomioCaratteristico(Matrice)::
  Restituisce il http://en.wikipedia.org/wiki/it:Polinomio_caratteristico[polinomio caratteristico] della matrice
  indicata.

[EXAMPLE]
====

`++PolinomioCaratteristico({{1,2},{3,4}})++` restituisce stem:[x^2-5x-2].

====

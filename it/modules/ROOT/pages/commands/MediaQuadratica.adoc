:page-en: commands/RootMeanSquare

MediaQuadratica(Lista di numeri)::
  Calcola la https://it.wikipedia.org/Media_(statistica)#Media_di_potenza[media quadratica] della lista di numeri
  indicata.

[EXAMPLE]
====

`++MediaQuadratica({3, 4, 5, 3, 2, 3, 4})++` restituisce 3.54

====

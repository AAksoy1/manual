= Comando DSYCampione
:page-en: commands/SampleSDY
ifdef::env-github[:imagesdir: /it/modules/ROOT/assets/images]

DSYCampione(Lista di punti)::
  Restituisce la https://it.wikipedia.org/Deviazione_standard[deviazione standard] campionaria delle ordinate dei punti appartenenti alla lista indicata.

[EXAMPLE]
====

`++DSYCampione({(2, 3), (1, 5), (3, 6), (4, 2), (1, 1), (2, 5)})++` restituisce _1.97_.

====

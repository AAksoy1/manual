== Sintassi CAS

PotenzaModulare(Numero, Numero, Numero)::
  Restituisce la potenza modulare dei numeri indicati.
  Per ulteriori informazioni consultare https://en.wikipedia.org/wiki/Modular_exponentiation[Modular exponentiation] (in
  inglese).

[EXAMPLE]
====

`++PotenzaModulare(5,12,13)++` restituisce stem:[1], poich√© stem:[mod(5^{12},13)=1].

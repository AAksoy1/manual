:page-en: tools/Input_Box
ifdef::env-github[:imagesdir: /nn/modules/ROOT/assets/images]

dialogboksen som kjem opp kan du skrive inn xref:/Namn_og_objekttekstar.adoc[objekttekst] og kva for objekt tekstfeltet
skal høyre til.

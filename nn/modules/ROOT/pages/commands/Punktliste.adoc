Punktliste[ <Liste med to-element lister> ]::
  Lagar ei liste med punkt frå ei liste som inneheld lister med to element.

[EXAMPLE]
====

*Døme:* `++Punktliste[{{1, 2}, {3, 4}}]++` gjev _\{(1, 2), (3, 4)}_.

====

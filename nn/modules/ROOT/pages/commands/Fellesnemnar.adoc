:page-en: commands/CommonDenominator

Fellesnemnar[ <Uttrykk>, <Uttrykk> ]::
  Finn den lågaste fellesnemnaren til dei to uttrykka.

[EXAMPLE]
====

*Døme:*

`++Fellesnemnar[3 / (2 x + 1), 3 / (4 x^2 + 4 x + 1)]++` gjev _4 x^2^ + 4 x + 1_.

====

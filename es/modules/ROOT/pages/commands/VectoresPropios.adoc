== Sintaxis CAS

VectoresPropios( <Matriz> )::
  Halla los vectores propios de la matriz dada.

[EXAMPLE]
====

`++VectoresPropios({{1, 2}, {3, 4}})++` devuelve stem:[ \left(\begin\{array}\sqrt\{33} - 3&-\sqrt\{33} -
3\\6&6\\\end\{array}\right) ]

====

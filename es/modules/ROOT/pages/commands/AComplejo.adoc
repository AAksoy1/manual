= Comando AComplejo
:page-en: commands/ToComplex_Command
ifdef::env-github[:imagesdir: /es/modules/ROOT/assets/images]

AComplejo( <Vector> )::
  Transforma un vector o un punto en un número complejo expresado algebraicamente.

[EXAMPLE]
====

`++AComplejo({3, 2})++` devuelve _3 + 2ί_.

====

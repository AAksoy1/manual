Suprime( <Lista>, <Lista> )::
  Suprime objetos de la primera lista cada vez que aparecen en la segunda lista.

[EXAMPLE]
====

`++ Suprime({1,3,4,4,9},{1,4,5})++` devuelve la lista `++{3,4,9}++`.

====

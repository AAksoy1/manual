:page-en: commands/SetLineStyle_Command

EstiloTrazo( <Línea>, <Número> )::
  Establece el estilo de línea del objeto dado según la tabla (números fuera de rango se consideran 0).

[cols=",",options="header",]
|===
|Número |Estilo
|0 |Pleno
|1 |Rayado largo
|2 |Rayado corto
|3 |Punteado
|4 |Punto-Raya
|===

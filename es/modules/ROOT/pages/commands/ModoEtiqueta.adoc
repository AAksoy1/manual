ModoEtiqueta( <Objeto>, <Número> )::
  Cambia el modo de etiquetar al objeto según el número asignado, de acuerdo con la siguiente tabla. Los valores enteros
  no admitidos, serán tratados como 0.

[cols=",",options="header",]
|===
|Número |Modo
|0 |Nombre
|1 |Nombre + Valor
|2 |Valor
|3 |Título
|===

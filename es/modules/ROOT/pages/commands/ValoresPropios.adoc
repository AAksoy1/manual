== Sintaxis CAS

ValoresPropios( <Matriz> )::
  Halla los valores propios de la matriz dada.

[EXAMPLE]
====

`++ValoresPropios({{1, 2}, {3, 4}})++` devuelve stem:[ \left\\{ \frac{\sqrt{33} + 5}\{2}, \frac{-\sqrt{33} + 5}\{2}
\right\} ]

====

== 什麼是視區

GeoGebra
提供不同的**視區**（View）來呈現動態連結的數學物件，當您在某個視區變更一個物件時，在其他視區中的此物件也會同步變更。

可以在您的裝置上自行調整各個**視區**的位置和大小，只要以滑鼠拖曳（5
版），或是先點選**檢視**選單，再拖曳視區右上角的圖示即可（6 版）。

GetTime[]::
GetTime[]::
  Returns a list such as _\{647, 59, 39, 23, 28, 2, 2011, "February", "Monday", 2}_, ie the current date and time in
  this order:
  milliseconds, seconds, minutes, hours (0-23), date, month (1-12), year, month (as text), day (as text), day (1 =
  Sunday, 2 = Monday, etc)

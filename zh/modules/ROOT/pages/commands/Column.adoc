Column[ <Spreadsheet Cell> ]::
Column[Spreadsheet Cell]::
  Returns the column of the cell as a number (starting at 1).

[EXAMPLE]
====
 `++Column[B3]++` gives you number _a = 2_ since column _B_ is the second column of the spreadsheet.

====

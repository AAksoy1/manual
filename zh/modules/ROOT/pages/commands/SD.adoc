== CAS 視窗

SD[ <List of Numbers> ]::
SD[ <List of Numbers> ]::
  Calculates the standard deviation of the numbers in the list.

[EXAMPLE]
====


`++SD[{1, 2, 3, 4, 5}]++` yields _1,41_.

====

== CAS Syntax

SD[ <List of Numbers> ]::
  Calculates the standard deviation of the numbers in the list.

[EXAMPLE]
====


`++SD[{1, 2, 3, 4, 5}]++` yields stem:[\sqrt\{2}].

====

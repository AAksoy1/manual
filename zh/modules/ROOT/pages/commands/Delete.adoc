= Delete 指令
:page-en: commands/Delete
ifdef::env-github[:imagesdir: /zh/modules/ROOT/assets/images]

Delete( <物件> )::
  刪除 <物件> 以及與它相依的所有物件。

[EXAMPLE]
====
 `++Delete(a)++` 刪除 _a_ 物件。

====

== 運算區語法

Delete( <物件> )::
  在 GeoGebra 中刪除 <物件> 以及與它相依的所有物件，並在運算區中移除指定給此物件的任何值。

[EXAMPLE]
====
 `++Delete(a)++` 刪除 _a_ 物件。

====

====

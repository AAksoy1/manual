Valec( <Kružnice>, <Výška> )::
  Vytvoří válec se podstavou základnou a zadanou výškou.

Valec( <Bod>, <Bod>, <Poloměr> )::
  Vytvoří válec se zadaným poloměrem a se zadanými body jako středy horní a dolní základny.

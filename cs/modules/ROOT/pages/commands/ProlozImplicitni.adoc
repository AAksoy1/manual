ProlozImplicitni( <Seznam bodů>, <Řád> )::
  Vyhledá nejlepší implicitní křivku řádu n ≥ 2, která prochází zadanými body. Pro výpočet je potřeba alespoň stem:[\frac{n(n+3)}2] bodů.

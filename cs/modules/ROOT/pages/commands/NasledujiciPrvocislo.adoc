NasledujiciPrvocislo( <Číslo> )::
  Určí nejmenší prvočíslo větší než zadané číslo.

[EXAMPLE]
====

`++NasledujiciPrvocislo[10000]++` dává výsledek _10007_.

====

[NOTE]
====

Viz také příkaz xref:/commands/PredchoziPrvocislo.adoc[PredchoziPrvocislo].

====

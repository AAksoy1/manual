Stred( <Kuželosečka> )::
  Vytvoří střed kružnice, elipsy nebo hyperboly.

[EXAMPLE]
====

`++Stred[x^2 + 4 y^2 + 2x - 8y + 1 = 0]++` vytvoří bod _A = (-1, 1)_

====

====

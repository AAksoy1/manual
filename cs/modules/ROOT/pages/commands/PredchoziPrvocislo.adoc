PredchoziPrvocislo( <Číslo> )::
  Určí největší prvočíslo menší než zadané číslo.

[EXAMPLE]
====

`++PredchoziPrvocislo[10000]++` určí _9973_.

====

[NOTE]
====

Viz též příkaz xref:/commands/NasledujiciPrvocislo.adoc[NasledujiciPrvocislo].

====

ZnakNaUnicode [ "<Znak>" ]::
  Výstupem příkazu je číslo daného znaku v https://en.wikipedia.org/wiki/cs:Unicode[tabulce Unicode].

[EXAMPLE]
====

`++ZnakNaUnicode["a"]++` dává jako výsledek číslo 97.

====

[NOTE]
====

Znak musí být zadán mezi uvozovkami.

====

====

NastavitStylCary( <Čára>, <Číslo> )::
  Změní styl zobrazení dané čáry podle následující tabulky (čísla mimo interval [0,4] jsou ignorována).

[cols=",",options="header",]
|===
|Číslo |Styl čáry
|0 |Plná
|1 |Čárkovaná dlouze
|2 |Čárkovaná krátce
|3 |Tečkovaná
|4 |Čerchovaná
|===

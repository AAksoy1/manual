AnalyzaRozptylu( <Seznam>, <Seznam>, ...)::
  Provede jednosměrný test http://en.wikipedia.org/wiki/Anova[ANOVA] na základě zadaných čísel.
  Výsledek je vydán jako seznam \{P value, F test statistic}.

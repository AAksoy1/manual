:page-en: commands/Numerator_Command

Citatel( <Funkce> )::
  Dostaneme čitatel dané funkce.

[EXAMPLE]
====

`++Citatel[(3x² + 1) / (2x - 1)]++` dostaneme _f(x) = 3x² + 1_.

====

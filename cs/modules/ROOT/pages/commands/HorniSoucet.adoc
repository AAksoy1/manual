= Příkaz HorniSoucet
:page-en: commands/UpperSum
ifdef::env-github[:imagesdir: /cs/modules/ROOT/assets/images]

HorniSoucet( <Funkce>, <Počáteční hodnota x>, <Koncová hodnota x>, <Počet obdélníků> )::
  Vypočítá horní součet zadané funkce na intervalu [_Počáteční hodnota x, Koncová hodnota x_] užitím předepsaného počtu
  obdélníků. Současně jsou tyto obdélníky i zobrazeny.

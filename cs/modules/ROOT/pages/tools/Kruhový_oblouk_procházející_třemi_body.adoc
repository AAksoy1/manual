= Nástroj Kruhový oblouk procházející třemi body
:page-en: tools/Circumcircular_Arc
ifdef::env-github[:imagesdir: /cs/modules/ROOT/assets/images]

také příkaz xref:/commands/KruhOblouk3Body.adoc[KruhOblouk3Body] ).

== GeoGebraウィンドウのリスト

複数の GeoGebra ウィンドウを開いている場合，このメニュー項目でウィンドウを切り替えることができる．

[NOTE]
====

GeoGebra をプレゼンテーションで，複数の GeoGebra
ファイルを同時に開いておき，それらを切り替えて使うときに役立つかもしれない．

= Cylinder コマンド
ifdef::env-github[:imagesdir: /ja/modules/ROOT/assets/images]

Cylinder( <円>, <高さ> )::
  与えられた底面と高さを持つ円柱を作成する．
Cylinder( <点>, <点>, <半径> )::
  与えられた半径と，上下の中心とする与えられた点を用いて円柱を作成する．

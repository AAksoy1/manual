= PrimeFactors コマンド
ifdef::env-github[:imagesdir: /ja/modules/ROOT/assets/images]

PrimeFactors( <数値> )::
  積が与えられた数と等しい http://en.wikipedia.org/wiki/ja:Prime_number[素数]のリストを返す ．

[EXAMPLE]
====

* `++PrimeFactors(1024)++` 出力： _\{2, 2, 2, 2, 2, 2, 2, 2, 2, 2}_.
* `++PrimeFactors(42)++` 出力： _\{2, 3, 7}_.

====

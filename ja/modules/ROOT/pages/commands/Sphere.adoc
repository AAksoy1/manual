= Sphere コマンド
ifdef::env-github[:imagesdir: /ja/modules/ROOT/assets/images]

Sphere( <点>, <半径> )::
  中心と半径で球面を作成する．
Sphere( <点>, <点> )::
  最初の点が中心で，２番目の点を通る球面を作成する．

[EXAMPLE]
====

`++Sphere((0, 0, 0), (1, 1, 1))++` 出力： _x² + y² + z² = 3_

====

== CAS での書式

Shuffle( <リスト> )::
  同じ要素をランダムな順序で並べたリストを返す．

[EXAMPLE]
====

* `++Shuffle({3, 5, 1, 7, 3})++` 出力例： _\{5, 1, 3, 3, 7}_.
* `++Shuffle(Sequence(20))++` は最初の20個の整数をランダムな順序で並べたものを返す．

====

= IsPrime コマンド
:page-en: commands/IsPrime
ifdef::env-github[:imagesdir: /ja/modules/ROOT/assets/images]

IsPrime( <数値> )::
  __数値__が素数である場合は _true_ ，そうでない場合は _false_ を返す．

[EXAMPLE]
====

* `++IsPrime(10)++` 出力： _false_,
* `++IsPrime(11)++` 出力： _true_.

====

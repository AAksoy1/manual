:page-en: commands/UnicodeToText

UnicodeToText( <整数のリスト> )::
  Unicode 数をテキストに変換する．

[EXAMPLE]
====

`++UnicodeToText({104, 101, 108, 108, 111})++` 出力：テキスト _"hello"_.

====

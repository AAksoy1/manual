= HarmonicMean コマンド
:page-en: commands/HarmonicMean
ifdef::env-github[:imagesdir: /ja/modules/ROOT/assets/images]

HarmonicMean( <数値のリスト> )::
  リスト要素の https://ja.wikipedia.org/%E8%AA%BF%E5%92%8C%E5%B9%B3%E5%9D%87[調和平均] を計算する．

[EXAMPLE]
====

`++HarmonicMean({13, 7, 26, 5, 19})++` 出力： _9.79_.

====

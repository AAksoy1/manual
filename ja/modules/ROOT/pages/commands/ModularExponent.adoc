== CAS での書式

ModularExponent( <数値>, <数値>, <数値> )::
  与えられた数値の冪剰余を返す．
  詳細はこちらを参照： https://en.wikipedia.org/wiki/ja:%E5%86%AA%E5%89%B0%E4%BD%99[冪剰余] ．

[EXAMPLE]
====

`++ModularExponent(5,12,13)++` は，stem:[1] を返す． 理由： stem:[mod(5^\{12},13)=1]．

====

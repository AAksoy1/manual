:page-en:commands/LinearEccentricity

LinearEccentricity( <円錐曲線> )::
  円錐曲線の線形離心率（半焦点間隔）を計算する．
  楕円や双曲線の場合,このコマンドは円錐曲線の中心とその一つの焦点との距離を返す.円の場合は0を返す.放物線の場合は，焦点と頂点との距離を返す．

[EXAMPLE]
====

`++LinearEccentricity(4x^2 - y^2 + 16x + 20 = 0)++` 出力： 2.24

====

= Divisors コマンド
:page-en: commands/Divisors
ifdef::env-github[:imagesdir: /ja/modules/ROOT/assets/images]

Divisors( <数値> )::
  数値自身を含む数値のすべての正の約数の数を計算する．

[EXAMPLE]
====

`++Divisors(15)++` 出力： _4_,　 _15_ の正の約数の個数（__15__を含む）．

====

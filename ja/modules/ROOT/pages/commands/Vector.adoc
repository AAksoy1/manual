= Vector コマンド
ifdef::env-github[:imagesdir: /ja/modules/ROOT/assets/images]

Vector( <点> )::
  与えられた点の位置ベクトルを返す．
Vector( <始点>, <終点> )::
  _始点_ から _終点_ へのベクトルを作成する．

[EXAMPLE]
====

`++Vector((1, 1), (3, 4))++` 出力： _u = stem:[\begin{pmatrix}2\\3\end{pmatrix}]_．

====

[EXAMPLE]
====

`++Vector((3, 2))++` 出力： _u = stem:[\begin{pmatrix}3\\2\end{pmatrix}]_．

====

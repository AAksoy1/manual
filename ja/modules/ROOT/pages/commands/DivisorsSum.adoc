= DivisorsSum コマンド
ifdef::env-github[:imagesdir: /ja/modules/ROOT/assets/images]

DivisorsSum( <数値> )::
  数値自身を含む数値のすべての正の約数の和を計算する．

[EXAMPLE]
====

`++DivisorsSum(15)++` 出力： _24_, 約数の和__1 + 3 + 5 + 15__.

====

InverseFDistribution( <分子自由度>, <分母自由度>, <確率> )::
  確率 _p_ における
  https://ja.wikipedia.org/F%E5%88%86%E5%B8%83[F分布]の累積分布関数の逆関数を計算する．_自由度_ で定義される．
  すなわち，_P(X ≤ t) = p_ となるような _t_ を求める．ここで ，_X_ はF分布に従う確率変数である．
  確率 _p_ は [0,1] の範囲でなければならない．

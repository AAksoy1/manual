= Parabola コマンド
:page-en:commands/Parabola
ifdef::env-github[:imagesdir: /ja/modules/ROOT/assets/images]

Parabola( <点>, <直線> )::
  焦点と直線を準線とする放物線を返す．

[EXAMPLE]
====

a = Line((0,1), (2,1))とするとき， `++Parabola((3, 3), a)++` 出力 _x² - 6x - 4y = -17_ .

====

====

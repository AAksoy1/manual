Slope( <直線> )::
  与えられた直線の傾きを返す．

[NOTE]
====

このコマンドは，傾きを表す三角形も作成する．この大きさは xref:/プロパティダイアログ.adoc[プロパティダイアログ]の
_スタイル_ のタブで変えることができる．

====

====

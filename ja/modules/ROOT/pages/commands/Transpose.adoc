= Transpose コマンド
ifdef::env-github[:imagesdir: /ja/modules/ROOT/assets/images]

Transpose( <行列> )::
  与えられた行列を転置する．

[EXAMPLE]
====

`++Transpose({{1, 2, 3}, {4, 5, 6}, {7, 8, 9}})++` 出力：行列 stem:[\begin\{pmatrix}1&4&7\\ 2&5&8\\
3&6&9\end\{pmatrix}]．

====

== CAS での書式

Transpose( <行列> )::
  与えられた行列を転置する．

[EXAMPLE]
====

`++Transpose({{a, b}, {c, d}})++` 出力：行列stem:[\begin\{pmatrix}a&c\\b&d\end\{pmatrix}]．

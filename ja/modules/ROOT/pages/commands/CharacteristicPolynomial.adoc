== CAS での書式

CharacteristicPolynomial( <行列> )::
  与えられた行列のhttps://en.wikipedia.org/wiki/ja:%E5%9B%BA%E6%9C%89%E5%A4%9A%E9%A0%85%E5%BC%8F[固有多項式] を返す．

[EXAMPLE]
====

`++CharacteristicPolynomial({{1,2},{3,4}})++` 出力： stem:[x^2-5x-2].

====

== CAS での書式

Rationalize( <数値> )::
  指定された _数_ の分数を作成し，分母に平方根が含まれている場合は，それを有理化する．

[EXAMPLE]
====

* `++Rationalize(3.5)++` 出力： stem:[\frac{7}\{2}].
* `++Rationalize(1/sqrt(2))++` 出力： stem:[\frac{\sqrt{2} }\{2}].

====

[NOTE]
====

こちらも参照： xref:/commands/MixedNumber.adoc[MixedNumber コマンド]．

====

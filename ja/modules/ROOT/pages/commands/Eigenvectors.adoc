== CAS での書式

Eigenvectors( <Matrix> )::
  与えられた行列の固有ベクトルを返す．

[EXAMPLE]
====

`++Eigenvectors({{1, 2}, {3, 4}})++` 出力： stem:[ \left(\begin\{array} +\sqrt{33} - 3&-\sqrt{33} -
3\\6&6\\\end\{array}\right) ]

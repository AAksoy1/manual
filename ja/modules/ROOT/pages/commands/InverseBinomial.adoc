InverseBinomial( <試行回数>, <成功確率>, <累積確率> )::
  _P(X ≤ n) ≥ p_ を満たす最小の整数 _n_ を返す．ここで，_p_ は累積確率であり，_X_ は _試行回数_ と _成功確率_
  によって定義される https://ja.wikipedia.org/wiki/%E4%BA%8C%E9%A0%85%E5%88%86%E5%B8%83[二項分布] の確率変数である．

  与えられた式の https://ja.wikipedia.org/%E9%99%B0%E9%96%A2%E6%95%B0#.E9.99.B0.E5.87.BD.E6.95.B0.E5.BE.AE.E5.88.86[陰関数の微分]

== CAS での書式

ImplicitDerivative( <f(x, y)> )::
  与えられた式の
  https://ja.wikipedia.org/%E9%99%B0%E9%96%A2%E6%95%B0#.E9.99.B0.E5.87.BD.E6.95.B0.E5.BE.AE.E5.88.86[陰関数の微分]
  を求める．

[EXAMPLE]
====

`++ImplicitDerivative(x + 2 y)++` 出力： _-stem:[\frac{1}{2}]_.

====

ImplicitDerivative( <式>, <従属変数>, <独立変数> )::
  与えられた式の
  https://ja.wikipedia.org/%E9%99%B0%E9%96%A2%E6%95%B0#.E9.99.B0.E5.87.BD.E6.95.B0.E5.BE.AE.E5.88.86[陰関数の微分]
  を求める．

[EXAMPLE]
====

`++ImplicitDerivative(x^2 + y^2, y, x)++` 出力： _-stem:[\frac{x}{y}]_.

====

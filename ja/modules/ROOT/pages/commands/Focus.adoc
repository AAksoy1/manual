= Focus コマンド
ifdef::env-github[:imagesdir: /ja/modules/ROOT/assets/images]

Focus( <円錐曲線> )::
  円錐曲線の（すべての）焦点を求める．

[EXAMPLE]
====

`++Focus(4x^2 - y^2 + 16x + 20 = 0)++` は，与えられた双曲線の2つの焦点を返す: __A__=(-2, -2.24) ， __B__=(-2, 2.24).

====

= RandomNormal コマンド
:page-en:commands/RandomNormal
ifdef::env-github[:imagesdir: /ja/modules/ROOT/assets/images]

RandomNormal( <平均>, <標準偏差> )::
  与えられた平均と標準偏差を持つ正規分布からランダムな数を生成する．

[EXAMPLE]
====

`++RandomNormal(3, 0.1)++` は，平均が _3_ で，標準偏差が _0.1_ の正規分布からランダムな値を返す．

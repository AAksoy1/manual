[NOTE]
====

image:16px-Menu_view_graphics.svg.png[Menu view
graphics.svg,width=16,height=16]xref:/グラフィックスビュー.adoc[グラフィックスビュー]），開きを管理する．
* 立方体の場合，作成される展開図はラテン十字である．
* こちらも参照： xref:/commands/Net.adoc[Net] コマンド．

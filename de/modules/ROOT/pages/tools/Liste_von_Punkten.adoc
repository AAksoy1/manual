xref:/Tabellen_Ansicht.adoc[Tabellen-Ansicht] eine Gruppe von Zellen aus, wo jeweils zwei Zellen entlang der Zeilen oder

die Liste von Punkten benennen, bearbeiten und erzeugen können.

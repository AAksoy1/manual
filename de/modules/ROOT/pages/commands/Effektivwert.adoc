:page-en: commands/RootMeanSquare

Effektivwert( <Liste von Zahlen> )::
  Liefert den https://de.wikipedia.org/wiki/Effektivwert[Effektivwert] der angegebenen Liste von Zahlen.

[EXAMPLE]
====

`++Effektivwert[{3, 4, 5, 3, 2, 3, 4}]++` ergibt _3.55_.

====

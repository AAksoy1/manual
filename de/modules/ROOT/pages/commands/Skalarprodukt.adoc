== CAS Syntax

Skalarprodukt( <Vektor>, <Vektor> )::
  Berechnet das Skalarprodukt zweier Vektoren.

[EXAMPLE]
====

Die Eingabe `++Skalarprodukt[{1, 3, 2},{0, 3, -2}]++` ergibt _5_.

====

[NOTE]
====

Siehe auch Befehl xref:/commands/Kreuzprodukt.adoc[Kreuzprodukt].

====

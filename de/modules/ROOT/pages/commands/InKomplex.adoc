:page-en: commands/ToComplex_Command

InKomplex( <Vektor> )::
  Verwandelt einen Vektor oder Punkt in eine komplexe Zahl in algebraischer Form.

[EXAMPLE]
====

`++InKomplex[(3, 2)]++` liefert _3 + 2ί_.

====

:page-en: commands/Determinant_Command

Determinante( <Matrix> )::
  Berechnet die Determinante der Matrix.

[EXAMPLE]
====

Die Eingabe `++Determinante[{{1, 2}, {3, 4}}]++` berechnet die Zahl _a = -2_.

====

Median( <Liste von Rohdaten> )::
  Berechnet den Median der angegebenen Zahlen in der Liste.

[EXAMPLE]
====

* `++Median[{1, 2, 3}]++` ergibt _2_.
* `++Median[{1, 1, 8, 8}]++` ergibt _4.5_.

====

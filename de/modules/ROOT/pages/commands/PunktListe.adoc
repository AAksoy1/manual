= PunktListe (Befehl)
:page-en: commands/PointList_Command
ifdef::env-github[:imagesdir: /de/modules/ROOT/assets/images]

PunktListe( <Liste> )::
  Erzeugt aus einer Liste von zweielementigen Listen eine Liste von Punkten.

[EXAMPLE]
====

`++PunktListe[{{1,2},{3,4}}]++` liefert \{(1,2),(3,4)}.

====

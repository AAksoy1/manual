Dimensjon[ <Vektor> ]::
  Gir dimensjonen til vektoren.
Dimensjon[ <Matrise> ]::
  Gir dimensjonen til matrisen.

[EXAMPLE]
====

`++Dimension[{{1, 2}, {3, 4}, {5, 6}}]++` gir _\{3, 2}_.

====

[EXAMPLE]
====

`++Dimensjon[{1, 2, 0, -4, 3}]++` gir _5_.

====

ForrigePrimtall[ <Tall a> ]::
  Gir det største primtallet som er mindre enn _a_.

[EXAMPLE]
====

`++ForrigePrimtall[10000]++` gir _9973_.

====

[NOTE]
====

*Merk:*

Se også kommandoen xref:/commands/NestePrimtall.adoc[NestePrimtall].

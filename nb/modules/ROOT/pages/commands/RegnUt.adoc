RegnUt[ <Funksjon> ]::
  Multipliserer ut funksjonsuttrykket.

[EXAMPLE]
====

`++RegnUt[(2 x - 1)^2 + 2 x + 3]++` returnerer stem:[4 x^2 - 2 x + 4].

====

== CAS-delen

RegnUt[ <Uttrykk> ]::
  Multipliserer ut uttrykket.

[EXAMPLE]
====

`++RegnUt[(2 x - 1)^2 + 2 x + 3]++` returnerer stem:[4 x^2 - 2 x + 4].

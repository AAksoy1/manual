NestePrimtall[ <Tall a> ]::
  Gir det minste primtallet som er større enn _a_.

[EXAMPLE]
====

`++NestePrimtall[10000]++` gir _10007_.

====

====

Se også kommandoen xref:/commands/ForrigePrimtall.adoc[ForrigePrimtall].

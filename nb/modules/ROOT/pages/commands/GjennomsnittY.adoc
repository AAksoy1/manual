GjennomsnittY[ <Liste med punkt> ]::
  Finner gjennomsnittet av _y_-koordiantene til punktene i listen.

[EXAMPLE]
====

`++GjennomsnittY[{(1, 2), (2, 3), (3, 4)}]++` gir 3 som er gjennomsnittet av 2, 3 og 4.

Fellesnevner[ <Uttrykk>, <Uttrykk> ]::
  Finner den minste fellesnevneren til de to utrykkene.

[EXAMPLE]
====

`++Fellesnevner[3 / (2 x + 1), 3 / (4 x^2 + 4 x + 1)]++` gir _4 x^2^ + 4 x + 1_.

====

Nevner[ <Funksjon> ]::
  Returnerer nevneren til funksjonen.

[EXAMPLE]
====

`++Nevner[5 / (x^2 + 2)]++` gir _f(x) = x^2^ + 2_.

====

== CAS-delen

Nevner[ <Uttrykk> ]::
  Returnerer nevneren til det rasjonale tallet eller uttrykket.

[EXAMPLE]
====

`++Nevner[5 / 3 + 2 / 5]++` gir _15_.

====

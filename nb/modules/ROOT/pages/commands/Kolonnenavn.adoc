Kolonnenavn[ <Cellenavn> ]::
  Returnerer navnet til kolonnen som cellen tilhører som en tekst.

[EXAMPLE]
====

`++Kolonnenavn[B3]++` gir teksten _“B”_ i xref:/Grafikkfelt.adoc[grafikkfeltet].
